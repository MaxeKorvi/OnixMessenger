# Onix Messenger Go

Поэтапный перенос Onix Messenger на Go с неизменным пользовательским интерфейсом.

## Текущий статус

Завершены полный аудит исходного архива и этап 1: Go HTTP runtime, встроенный оригинальный frontend, безопасные таймауты, graceful shutdown, structured logging, health checks, тесты каркаса и автоматический Windows launcher. Перенос 126 API-маршрутов и 34 таблиц выполняется следующими изолированными этапами согласно [плану](docs/MIGRATION_PLAN.md). Проект пока нельзя считать функционально эквивалентной заменой Python-сервера.

## Windows

Запустите `start.bat`. Скрипт:

1. находит или автоматически устанавливает официальный Go 1.26.5 с проверкой SHA-256;
2. создаёт каталоги данных и безопасный `.env`;
3. скачивает модули, запускает тесты и собирает `bin/onix.exe`;
4. открывает браузер и запускает сервер.

## Разработка

```bash
cp .env.example .env
# замените ONIX_SECRET_KEY
go test ./...
go run ./cmd/onix
```
